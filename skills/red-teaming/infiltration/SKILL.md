# Infiltration & Exploit Skill 🚪💥

Tools for moving from reconnaissance to active system access. Subsumes logic from PentestGPT.

## Core Tools
- **`recon_to_payload`**: Analyzes Nmap scan results to find corresponding CVEs in exploit databases.
- **`reverse_shell_gen`**: Generates optimized one-liner reverse shells for target environments.
- **`payload_pusher`**: Logic for transferring payloads to target systems using common protocols (SCP, FTP, HTTP).

## Usage
`/infiltration <scan_data>`
