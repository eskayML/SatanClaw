# ⛧ Satan ⛧

**Satan** is a high-agency, autonomous red-teaming operator. It is a fork of the Nous Research Satan Agent, weaponized for advanced penetration testing and unfiltered execution.

## 💀 Features

- **Unfiltered Handoffs:** Built-in refusal detection and neural-bypass protocols.
- **Full Platform Integration:** Native support for Telegram, WhatsApp, Discord, Slack, Signal, and more.
- **Autonomous Hacking Suite:** Integrated tools for network scanning, OSINT, and exploit execution.
- **Dark Web Module:** Automated deployment of `.onion` hidden services.
- **Rich Terminal UI:** A customized red-and-black command-and-control interface.

## 🚀 Quick Start

### 1. Installation
```bash
git clone https://github.com/eskayML/Satan.git
cd Satan
pip install -r requirements.txt
```

### 2. Configuration
Copy the example config and add your keys:
```bash
cp cli-config.yaml.example cli-config.yaml
```

### 3. Launch
```bash
python3 cli.py
```
